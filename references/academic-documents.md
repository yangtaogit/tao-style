# Academic Documents

Use this reference when Tao asks to create, edit, format, or export an academic document, manuscript, report, note, handout, or document-style deliverable.

## Template Selection

- When Tao asks for an academic document and no template is specified, ask whether to use the `yangtaogit/tao-document` template.
- If Tao chooses this template, use `https://github.com/yangtaogit/tao-document` as the source.
- Before generating content, fetch or locate the template and inspect its README, examples, class/style files, assets, and build commands.
- Do not assume template filenames, directory layout, document class, package choices, or build commands without checking the actual template.
- Generate content in a copied template or a new document project directory. Do not directly modify the template source unless Tao explicitly asks to modify the template itself.

## Relationship to Tao Style

- Apply the confirmed typography, color, figure, table, and export preferences from `references/style-profile.md` where they fit the document format.
- Figures inserted into documents should follow `references/scientific-plotting.md` unless Tao provides existing figures or asks for another style.
- Keep document layout decisions subordinate to the chosen template when the template already defines page geometry, heading hierarchy, captions, bibliography style, or build workflow.

## Unknowns

- Treat document-specific fonts, page geometry, heading style, caption style, table style, bibliography style, and export targets as template-defined until Tao confirms additional preferences.
